# htmlcssts_challenge
Im Ordner 01 TS Projekte stehen 3 mögliche Projekte zur Auswahl. Du kannst dir ein Projekt auswählen und dies anschliessend umsetzen.

Versucht die Nutzung von AI auf ein Minimum zu reduzieren und geht Schrittweise voran mit der Entwicklung:

    1. Erstellt euch ein Simples Wireframe/Mockup wie eure Applikation aussehen könnte.
    2. Implementiert Funktionen schrittweise, mit Basisfunktionen anfangen und dann Schritt für Schritt an die Visuellen Elemente.
    3. Erstellt ein simples Backend anhand dem gelerntem vom 295er ÜK welcher die randomnumber API und Logik ersetzt, so dass das Frontend nur noch effektiv Daten darstellen muss.
